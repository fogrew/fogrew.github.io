Сверстал и запрограммировал систему проверки переходов по рекламным ссылкам.
Валидная, кроссбраузерная, не адаптивная вёрстка. html, css3.
Бэк на php (5 версии), с авторизацией и CRUD'ом.

Она была закрыта паролем, поэтому только скриншот версии под правами user (у admin ещё кнопочки управления были).
